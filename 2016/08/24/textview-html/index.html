如何让TextView支持Html

发表于 2016-08-24   |   阅读次数   |  

大家可能都知道webview–一个在android上支持html的控件.比较重量级的.可能有时候我们只想让我们的一个TextView支持html.这个时候就需要android原生类Html了.调用Html.from()方法.会去解析html文本,并返回一个Spanned.但是毕竟是轻量级的,所以只对部分标签提供了支持.以下是源码:


    private void handleStartTag(String tag, Attributes attributes) {

我遇到的问题就是处理span标签,所以需要自己处理span标签.例如如下的一个span标签.

    <span style=”color: rgb(0, 112, 192);”>（DT君注：纽约目前很难找到公开招聘的网络直播这类职业</span>

首先我们要获取标签内的文本,这个比较容易,通过output这个参数即可以获取,其次我们要获取style属性,但是我们发现attributes这个参数并没有传递下来.最后在网上找到了一个解决办法,通过反射去获取属性值,亲测可以.上代码


Field elementField = xmlReader.getClass().getDeclaredField(“theNewElement”);//“theElement”固定写法
elementField.setAccessible(true);
Object element = elementField.get(xmlReader);
Field attsField = element.getClass().getDeclaredField(“theAtts”);//“theAtts”固定写法
attsField.setAccessible(true);
Object atts = attsField.get(element);
Field dataField = atts.getClass().getDeclaredField(“data”);//“data”固定写法
dataField.setAccessible(true);
String[] data = (String[])dataField.get(atts);
Field lengthField = atts.getClass().getDeclaredField(“length”);//“length”固定写法
lengthField.setAccessible(true);
int len = (Integer)lengthField.get(atts);
String attrStr = null;
for(int i = 0; i < len; i++) {
    if(“style”.equals(data[i  5 + 1])) {//①”style”自己的属性名
         attrStr = data[i  5 + 4];
     }
}

上段代码中只需要改写把①中的属性名改写为自己的属性名即可获取到属性值.我得到的结果就是:

attrStr = color: rgb(0, 112, 192)

到这儿,我们已经获取到了文本及颜色值接下来就用ForegroundColorSpan给文字设置颜色

output.setSpan(new ForegroundColorSpan(color),start,end, Spanned.SPAN_EXCLUSIVE_EXCLUSIVE);
