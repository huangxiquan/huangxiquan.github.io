Python-aiohttp搭建简单服务器

发表于 2016-08-15   |   阅读次数   |  

前言

用aiohttp搭建一个简单服务器,输出Hello World.代码不超过10行.

环境配置

  • 安装python3
  • 安装第三方库aiohttp
    pip3 install aiohttp

编写代码

总共代码不超过10行即可搭建一个简易服务器.

from aiohttp import web
def hello(request) :
        return web.Response(body=b”Hello World”)
app = web.Application()
app.router.add_route(‘GET’,’/‘,hello)
web.run_app(app)

运行python文件,然后打开浏览器访问即可.
我们可以去看看,里面有简单的介绍原理.我这里也简单的说一下.

代码解释

handler是一个有且只有一个Request类型的参数,并返回Response类型的函数.说简单点,handler就是处理请求的方法,接收一个请求并给出响应.类似于java中的servlet. 


def hello(request) :
        return web.Response(body=b”Hello World”)

我们现在有了处理请求的方法,但是当请求被发送到服务器,服务器是怎么知道要用哪个handler处理呢?所以还需要一步,就是handler注册.我们需要将path(访问路径)和handler对应起来.

app = web.Application()
app.router.add_route(‘GET’,’/‘,hello)

Application的生命周期是服务运行期间,所以会在路由表保存在Application里面.当有请求到来时,会根据访问路径查找路由表,然后调用对应handler.例子中的路径就是根目录.请求方式是GET.
最后就是将服务跑起来,没什么好解释的.
web.run_app(app)
